# 🔮 Project Astro

AI-powered astrology web app that generates kundli charts and provides personalized insights using LLMs.

---

🔗 Live Demo: Coming Soon

---

## 🚀 Features

- Kundli (birth chart) generation  
- Lagna & Moon chart visualization  
- AI astrology chat assistant  
- Firebase backend integration  
- Gemini API integration  
- Hindi + English support
